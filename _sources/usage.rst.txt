Usage
=============

Installation
---------------
GEMAct can be easily installed via ``pip``:

   pip install gemact

In case you are installing GEMAct within Google Colab, please make sure to restart the kernel or to refresh the page before using it.

Tutorials
---------------

Users might be interested in the following Jupyter notebooks available on `GitHub <https://github.com/gpitt71/gemact-code/tree/main/vignette>`_.
We provide several examples to familiarize with GEMAct functionalities. The folder contains:

- The replication code of the manuscript *GEMAct: an actuarial Python package for (re)insurance modelling*.
- The supplementary material of the manuscript *GEMAct: an actuarial Python package for (re)insurance modelling*.
