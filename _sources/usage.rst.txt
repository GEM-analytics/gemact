Usage
=============

In order to install GEMAct::

   pip install gemact

In case you are installing the GEMAct package in Google Colab, please make sure restart the kernel or to refresh the page before using GEMAct.

Tutorials
---------------

Practitioners who want to start using GEMAct might be interested in the Jupyter notebooks we have available on `GitHub <https://github.com/gpitt71/gemact-code/tree/main/vignette>`_ .
We provide examples on all GEMAct functionalities.

* Click `here <https://github.com/gpitt71/gemact-code/blob/main/vignette/lossreserve.ipynb>`_ to understand how to compute loss reserves in GEMAct.
* Click `here <https://github.com/gpitt71/gemact-code/blob/main/vignette/lossaggregation.ipynb>`_ to understand how to perform risks aggregation in GEMAct.
* Click `here <https://github.com/gpitt71/gemact-code/blob/main/vignette/copulas.ipynb>`_ to understand how to use the copula functions supported in GEMAct.
