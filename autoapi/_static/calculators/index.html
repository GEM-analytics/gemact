static fast_fourier_transform(severity, frequency, n_aggr_dist_nodes, discr_step, tilt, tilt_value)#

Aggregate loss distribution via Fast Fourier Transform.

Parameters

  • severity (dict) – discretized severity, nodes sequence and discrete probabilities.

  • frequency (Frequency) – frequency model (adjusted).

  • n_aggr_dist_nodes (int) – number of nodes in the approximated aggregate loss distribution.

  • discr_step (float) – severity discretization step.

  • tilt_value (float) – tilting parameter value of FFT method for the aggregate loss distribution approximation.

  • tilt (bool) – whether tilting of FFT is present or not.

Returns

aggregate loss distribution empirical pmf, cdf, nodes

Return type

dict

static upper_discr_point_prob_adjuster(severity, deductible, exit_point, discr_step)#

Probability of the discretization upper point in the local moment. In case an upper priority on the severity is provided, the probability of the node sequence upper point is adjusted to be coherent with discretization step size and number of nodes.

Parameters

  • severity (Severity) – severity model.

  • deductible (int or float) – deductible, also referred to as retention or priority.

  • exit_point – severity ‘exit point’, deductible plus cover.

  • discr_step (float) – severity discretization step.

Returns

probability mass in (u-d/h)*m

Return type

numpy.ndarray
