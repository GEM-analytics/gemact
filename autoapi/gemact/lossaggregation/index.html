class LossAggregation(copula, copula_par, margins, margins_pars, **kwargs)[source]#

Class representing the sum of positive countinuous random variables. Dependence structure is specified by a copula and a set of given marginals.

Parameters

  • copula (str) – Name of the copula that describes the dependence structure.

  • copula_par (dict) – Parameters of the copula.

  • margins (list) – List of the marginal distributions.

  • margins_pars (list) – List of the marginal distributions parameters. It must be a list of dictionaries.

  • **kwargs – See below

Keyword Arguments
  • random_state (int) –

    Random state for the random number generator in MC.

  • sample_size (int) –

    Number of simulations of Monte Carlo (mc) method.
